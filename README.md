# Retail Sales & Campaign Performance Dashboard (Power BI)
- # Sales-Campaign-Performance---OCT-2024
- 
## Overview
This project analyzes retail campaign performance across multiple states for October 2025. The dashboard provides insights into sales targets, achievements, store performance distribution, and payout analysis.

## Key Metrics
- Total Stores
- Sales Target vs Achievement
- Achievement Percentage
- Store Performance Distribution
- Campaign Payout Analysis

## Tools Used
- Power BI
- Excel
- Data Modeling (DAX)

## Dashboard Insights
- State-wise sales performance comparison
- Target vs Achievement analysis
- Store achievement distribution
- Campaign payout tracking

## Files
- `Retail_Sales_Campaign_Performance_Oct_2025.pbix` → Power BI dashboard file
- `dataset.xlsx` → Anonymized dataset used for analysis
